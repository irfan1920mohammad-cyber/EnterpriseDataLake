About Enterprise Data Lake

A Secure Multi-Tenant Dataset Management Platform

Manage Enterprise Data Securely

Enterprise Data Lake is a modern web application developed using Django, Python and Bootstrap. It allows organizations and students to securely upload, validate, manage and download datasets.

Every user gets a completely isolated workspace. Uploaded datasets are protected using SHA-256 hashing and organized through an intelligent dashboard.
